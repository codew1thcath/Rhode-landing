# Rhode Skin — Landing Page

A concept landing page inspired by Rhode Skin, built as a frontend portfolio project. This project demonstrates UI/UX implementation skills using only HTML and CSS, with minimal vanilla JavaScript for interactions.

---

## Overview

This project is a fully responsive, single-page website recreating the aesthetic and layout of a modern skincare brand. It was built to showcase frontend development skills including layout design, typography, animation, and attention to visual detail.

> Note: This is a student/portfolio project and is not affiliated with or endorsed by Rhode Skin LLC or Hailey Bieber.

---

## Features

- Responsive layout across desktop and mobile viewports
- Custom animated cursor with smooth ring-follow effect
- Hero section with floating product illustration and staggered entrance animations
- Auto-scrolling marquee product strip
- Product cards with hover-reveal interactions
- Scroll-triggered reveal animations using IntersectionObserver
- Ingredient highlight section
- Customer testimonials
- Newsletter signup form
- Multi-column footer

## Design Decisions

**Typography** — Cormorant Garamond was chosen as the display typeface for its editorial, high-fashion character. Jost serves as the body font for its clean, modern readability.

**Color Palette** — A warm cream and blush palette was used to reflect the brand's soft, approachable identity while maintaining contrast for accessibility.

**Layout** — The hero uses a two-column split grid to establish visual hierarchy immediately. Generous whitespace throughout reinforces the brand's clean, luxury feel.

**Animation** — Animations are intentional and minimal. Entrance animations use staggered delays to guide the eye. The floating product illustration and scroll reveals add depth without distraction.
